## 七、部署计划

### 开发环境
- 前端：Vite开发服务器（端口5173）
- 后端：Go开发服务器（端口8888）
- 数据库：MySQL本地实例

### 生产环境
- 前端：
  - 构建：`npm run build`
  - 部署：Nginx静态文件服务
  - CDN：可选，用于加速静态资源

- 后端：
  - 构建：`go build -o career-api`
  - 部署：systemd或Docker
  - 反向代理：Nginx
  - 负载均衡：多实例（可选）

- 数据库：
  - MySQL主从复制（可选）
  - 定期备份
  - Redis缓存（可选）

### 监控和告警
- 日志收集：ELK Stack或Loki
- 性能监控：Prometheus + Grafana
- 错误追踪：Sentry（可选）
- 告警：邮件/钉钉/企业微信
