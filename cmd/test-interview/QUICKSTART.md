# 快速开始

## 一分钟启动面试模拟系统

### 1. 设置API密钥

```bash
export DEEPSEEK_API_KEY=sk-your-api-key-here
```

### 2. 运行程序

```bash
cd cmd/test-interview
./run.sh
```

### 3. 开始面试

```
🎤 欢迎使用面试模拟系统
📝 这是一个传统的打字问答面试系统
💡 输入 'quit' 或 'exit' 退出
==================================================

🤔 面试官: 请先做一个简单的自我介绍
👤 你: 我是一名计算机专业的学生...
```

输入 `quit` 或 `exit`：

```
👤 你: quit

👋 面试结束，感谢参与！
⏱️  面试时长: 3m15s
```

## 常见问题

### Q: 如何获取DeepSeek API密钥？

A: 访问 https://platform.deepseek.com/ 注册账号并获取API密钥。

### Q: 程序会保存我的面试记录吗？

A: 当前版本仅在内存中保存对话历史，程序退出后会丢失。

### Q: 可以自定义面试问题吗？

A: 可以修改代码中的 `systemPrompt` 变量来自定义面试官的行为。

## 技术特点

- ✅ 流式输出，实时显示AI回复
